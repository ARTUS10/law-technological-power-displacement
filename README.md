## What the code does

### Step 2.2 — Spain BOE Regulatory Index (net-change variant)

Constructs a weighted longitudinal index of regulatory capacity for Spain (1978–2024) from
62 regulatory bodies recorded in the Official State Gazette (*Boletín Oficial del Estado*, BOE).
The net-change variant (entries minus exits) addresses the critique that the cumulative
index rises mechanically. The script runs the central mechanistic test (T4) on both variants
and reports structural-stability (Chow) results across two sub-periods.

### Step 2.4 — EU-27 Panel Robustness Tests

Applies four robustness checks to the EU-27 panel drawn from the Digital Society Project v8
(Mechkova et al. 2026):

| Test | Purpose |
|------|---------|
| AR(1) correction | Rules out autocorrelation artefact |
| Random slopes | Country-level OLS confirms beta > 0 in 26/26 members |
| Granger causality | Directional test: phi -> dPsi vs reverse |
| Panel cointegration | ADF on residuals establishes long-run structural link |

## Data

### Spain BOE index
The regulatory body catalogue is embedded directly in `analysis.py` (list `ENTRIES_BOE` /
`EXITS_BOE`). Each record includes year, weight, sector, and archival description with the
corresponding BOE reference. No external file is required for Step 2.2.

### EU-27 panel (Digital Society Project v8)
The DSP data cannot be redistributed here per the terms of use. Download instructions:

1. Go to <https://digitalsocietyproject.org/data/>
2. Click **"Click here to download as CSV"**
3. Unzip the archive
4. Place the `.csv` file in the `data/` directory
5. Update the `DSP_FILE` variable at the top of `analysis.py` if the filename differs

Variables used:

| Variable | Role | Direction |
|----------|------|-----------|
| `v2smgovfilprc` | Sigma proxy (citizen freedom) | High = more filtering = less freedom; inverted in code |
| `v2smregcap` | Omega proxy (regulatory capacity) | High = more regulation; used directly |

## Reproducing the results

```bash
# Clone the repository
git clone https://github.com/ARTUS10/law-technological-power-displacement.git
cd law-technological-power-displacement

# Install dependencies
pip install -r requirements.txt

# Place the DSP CSV in data/ then run
python analysis.py
```

Output files appear in `figures/` and `results/`.

## Key variables and notation

| Symbol | Description |
|--------|-------------|
| Omega (Ω) | Accumulated effective public domain (regulatory weight) |
| Sigma (Σ) | Citizen rights space (normalised) |
| Psi (Ψ) | Asymmetry index: Omega / Sigma |
| phi (φ) | Fracture ratio: sufficient statistic of the dynamic system |
| dPsi | First difference of Psi (annual change) |
| T4 | Central mechanistic test: phi -> dPsi |
| T5 | Within-country (fixed-effects) estimator |
